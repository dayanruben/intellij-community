math
Newly available across major browsers

Chrome 109, Chrome Android 109, Edge 109, Firefox 4, Firefox Android 4, Opera 95, Safari 5.1, Safari iOS 5. Baseline since 2023

The <math> MathML element is the top-level MathML element, used to write a single mathematical formula. It can be placed in HTML content where flow content is permitted.

Note: See the Authoring MathML page for tips to properly integrate MathML formulas in your web pages and the Guides page for more demos.
