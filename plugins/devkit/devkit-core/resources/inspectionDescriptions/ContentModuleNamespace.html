Validates namespace attributes in content module declarations within plugin.xml files.

The inspection reports the following issues:

  • Missing namespace: Content namespace is required when plugin content includes at least one module with non-private visibility
  • Invalid namespace format: Namespace must match the required pattern ([a-zA-Z0-9]+([_-][a-zA-Z0-9]+)*) and its length must be between 5 and 30 characters
  • Inconsistent namespaces: All content elements must define the same namespace
